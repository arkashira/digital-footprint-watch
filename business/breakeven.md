# breakeven.md
## Unit Economics
### Cost per Active User
We estimate the following costs for each active user of the digital-footprint-watch product:
- Compute: $0.50 USD per user per month (assuming 10 minutes of daily compute time per user at $0.10 per minute)
- Storage: $0.25 USD per user per month (assuming 100MB of storage per user at $0.0025 per MB)
- Bandwidth: $0.10 USD per user per month (assuming 100MB of bandwidth per user at $0.001 per MB)
Total cost per active user: $0.85 USD per month

## Pricing Tiers
We propose the following pricing tiers for the digital-footprint-watch product:
### Tier 1: Basic
- Price: $9.99 USD per month
- Features: Limited to 1 platform monitoring, daily updates, and basic reporting
### Tier 2: Premium
- Price: $29.99 USD per month
- Features: Includes monitoring across 5 platforms, real-time updates, and advanced reporting
### Tier 3: Enterprise
- Price: $99.99 USD per month
- Features: Includes monitoring across 10+ platforms, real-time updates, advanced reporting, and dedicated support

## Customer Acquisition Cost (CAC) Range
Based on industry benchmarks, we estimate the CAC range for the digital-footprint-watch product to be between $10 USD and $50 USD per user.

## Lifetime Value (LTV) Estimate
Assuming an average user retention period of 12 months and an average revenue per user (ARPU) of $19.99 USD per month (based on the pricing tiers), we estimate the LTV to be:
LTV = ARPU x User Retention Period = $19.99 x 12 = $239.88

## Break-Even Analysis
To calculate the break-even point, we need to consider the CAC and LTV estimates.
Break-Even Point = CAC / (LTV / ARPU)
Assuming a CAC of $30 USD and an LTV of $239.88, we get:
Break-Even Point = $30 / ($239.88 / $19.99) ≈ 3.75 months

## Break-Even Users Count
To calculate the break-even users count, we need to consider the total cost per active user and the pricing tiers.
Break-Even Users Count = Total Monthly Costs / (Pricing Tier - Total Cost per Active User)
Assuming a total monthly cost of $1,000 USD and a pricing tier of $9.99 USD per month, we get:
Break-Even Users Count = $1,000 / ($9.99 - $0.85) ≈ 113 users

## Path to $10K MRR
To reach $10,000 USD in monthly recurring revenue (MRR), we need to acquire and retain a certain number of users across the pricing tiers.
Assuming a mix of 20% Basic, 60% Premium, and 20% Enterprise users, we get:
- Basic: 20% x $9.99 = $1.998 per user
- Premium: 60% x $29.99 = $17.994 per user
- Enterprise: 20% x $99.99 = $19.998 per user
Weighted Average Revenue per User (ARPU) = ($1.998 + $17.994 + $19.998) / 3 ≈ $13.33 per user
To reach $10,000 USD in MRR, we need:
$10,000 / $13.33 ≈ 750 users
Assuming a conversion rate of 10% from free trial to paid, we need:
750 users / 0.1 ≈ 7,500 free trial users

To achieve this, we can aim to acquire 250 users per month for 3 months on the Premium tier, which would contribute $7,497.50 USD to the MRR, and 50 users per month for 3 months on the Enterprise tier, which would contribute $2,499.50 USD to the MRR, and 100 users per month for 3 months on the Basic tier, which would contribute $998 USD to the MRR. This would put us on the path to reaching $10,000 USD in MRR.
